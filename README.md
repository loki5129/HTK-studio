# HTK-studio

## Machine Learning Tetris
Theme- Old is Gold
Team members- Thomas, Himika, Kamile

<p>The Machine Learned Tetris Game Server is a server‑based implementation of the classic Tetris game, enhanced with a machine learning (ML) opponent. This system manages game logic, communicates with a client interface, and enables users to play against either the system or an AI-driven Tetris player</p>

### System Overview
This system include: 
<li>A dedicated Ubuntu server running the game logic.</li>
<li>A client interface built in JavaScript, HTML, and CSS.</li>
<li>Real-time communication between client and server.</li>
<li>A machine learning opponent using a genetically optimized mathematical evaluation function.</li>

### Tech Stack

Frontend:
<li>HTML– Structure</li>
<li>CSS– Styling</li>
<li>JavaScript– Game interaction</li>
<br>
Backend:
<li>Node.js– Server runtime</li>
<li>Custom game logic– Tetromino mechanics, scoring, AI</li>
<li>Genetic algorithm– Used to train AI </li>
